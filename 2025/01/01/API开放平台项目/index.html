项目介绍

背景：

  1. 当前端需要后端的接口时
  2. 使用现成的系统功能（http://api.btstu.cn/)

考虑内容：

  1. 防止攻击（安全）
  2. 不能随便调用（限制，开通）
  3. 计费
  4. 流量保护
  5. API接入
